
# Site Builder SaaS — Documentação

Este repositório contém a documentação para implementar um Criador de Sites para Venda com:
- Editor no-code
- Templates completos + temas + blocos
- Multi-tenant + domínios com SSL
- SEO + mensuração (GA4/GTM/Ads/Pixels) com Consent Mode v2
- PageSpeed/Core Web Vitals como requisito
- Shop básico
- Billing recorrente + régua (dunning) via Evolution API v1
